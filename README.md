# Andrew's Utils

My own utils for my own use. I created this package so I didn't have to type the same code over and over

## Installation

```bash
npm i @andrewdragon/utils
pnpm i @andrewdragon/utils
```

## Usage

```ts
import { randomNumber } from "@andrewdragon/utils"
```
